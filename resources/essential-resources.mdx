This living document is a collection of outstanding guides, articles, and documentation aimed at enhancing the productivity and skills of programmers and designers. It will be updated periodically with new and insightful resources.

## Design Guidelines and UI/UX

- [Shopify Design Guidelines](https://shopify.dev/docs/apps/design-guidelines)
  - Comprehensive design principles and best practices for creating apps within the Shopify ecosystem.

- [Polaris - Shopify Design System](https://polaris.shopify.com/)
  - A detailed design system that helps create a consistent user experience for Shopify products.

## Coding Practices

- [Good Coding Practices by Qulix Systems](https://www.qulix.com/about/blog/good-coding-practices/)
  - An article that outlines practical advice for writing clean and maintainable code.

## Additional Recommended Resources

- [Google's Web Fundamentals](https://developers.google.com/web/fundamentals)
  - Google's best practices for building a modern, performant web experience.

- [Mozilla Developer Network (MDN) Web Docs](https://developer.mozilla.org/)
  - A premier resource for web developers, offering comprehensive documentation and learning materials for web technologies.

Remember, the journey of learning and improvement never ends. Keep this list handy as a gateway to evolving your craft.
